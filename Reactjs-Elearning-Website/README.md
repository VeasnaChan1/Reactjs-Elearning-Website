# 📚 EduReact — E-Learning Platform

A Coursera-inspired e-learning web application built with React and Vite. Users can browse a course catalog, watch video lessons, and track their learning progress through a personal dashboard.

> Built as a team project for a Front-End Development course.

## ✨ Features

- 🔍 **Course Catalog** — Browse and explore a list of available courses
- 🎬 **Video Lessons** — Watch embedded video content inside each course
- 📊 **Progress Dashboard** — Track enrolled courses and learning progress
- 🔀 **Multi-page Navigation** — Seamless routing between pages with React Router

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------|---|
| [React](https://react.dev/) | UI library |
| [Vite](https://vitejs.dev/) | Build tool & dev server |
| [React Router](https://reactrouter.com/) | Client-side routing |

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VeasnaChan1/Reactjs-Elearning-Website.git
   cd Reactjs-Elearning-Website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
